# Contributing

## Types

- Add

- Remove

- Update

- Refactor

- Bugfix

## Branch naming

Use the following format for branch names:

```txt
<type>/<short-description>
```

Example:

```txt
bugfix/lint-errors
```

## Commit guidelines

Use the following format for commit messages:

```txt
<type>: <summary>
```

Example:

```txt
Bugfix: Lint errors
```
