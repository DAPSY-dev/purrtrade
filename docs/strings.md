# Strings

## Structure

Strings are an object with the following structure:

```json
{
  "KEY": "Value"
}
```

Keys are in **SCREAMING_SNAKE_CASE** style.

## Special keys

- `__LANGUAGE__` - Current language.

  Example:

  ```json
  {
    "__LANGUAGE__": "English"
  }
  ```

  ```json
  {
    "__LANGUAGE__": "Български"
  }
  ```
