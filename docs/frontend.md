# Front-End

## Installation prerequisites

- Node.js: `v20.18.0`

## Installation

- Install dependencies;

## Scripts

- Install: `npm install`

- Start dev: `npm run dev`

- Test: `npm run test`

- Build: `npm run build`

- Lint: `npm run lint`

- Preview: `npm run preview`

- Favicon check: `npm run faviconCheck`

  - Start dev or build and preview before running favicon check
